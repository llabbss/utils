## 概述
    一个基于订阅发布封装的队列上传器
    目前仅支持单并发，多并发后续会支持
    axios简易封装
    上传器通用，React项目亦可使用，只是为了图方便使用了vue搭建

    
### 启动服务
    mock数据服务，可以自行修改
    ```bash
    cd server
    npm run server
    ```
### 启动前端
    ```bash
    npm run dev
    ```
